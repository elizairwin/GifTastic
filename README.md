# GifTastic

## Description: 
An interactive app that allows users to search for gifs. The user enters a Gif search and the page populates with 10 items related to the search. Code is included to stop/start the Gifs.

## Technology Used:
* HTML
* CSS
* Bootstrap
* JavaScript
* jQuery
